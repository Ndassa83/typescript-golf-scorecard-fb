# Golf Scorecard

## Description

Create custom players and courses to use for custom games and keep track of scores.
Users can instantaneously see statistics based on all previous round with the ability to sort by player, course, and date.


## Technologies

- Built the interface using React, TypeScript, and Google’s Material UI.
- Uses firebase for saving users, courses & historical statistics
- Hosted by Firebase Hosting
